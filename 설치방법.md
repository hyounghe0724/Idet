# 식단 트래커 안드로이드 홈 화면에 앱처럼 띄우기

이 5개 파일을 웹에 올리고, 폰 크롬에서 "홈 화면에 추가" 하면 아이콘이 깔리고 앱처럼 전체화면으로 실행됨.

파일 목록:
- `index.html` — 앱 본체
- `manifest.json` — 앱 정보(이름·아이콘·색)
- `sw.js` — 오프라인 캐시(인터넷 없어도 실행)
- `icon-192.png`, `icon-512.png` — 앱 아이콘

---

## 방법 A. GitHub Pages (무료, 추천)

**1. GitHub 저장소 만들기**
- github.com 로그인 → 우측 상단 `+` → `New repository`
- 이름 아무거나 (예: `diet-tracker`), Public 선택, `Create repository`

**2. 파일 5개 업로드**
- 만든 저장소에서 `Add file` → `Upload files`
- 위 5개 파일을 드래그해서 전부 올림
- 아래 `Commit changes` 클릭

**3. Pages 켜기**
- 저장소 `Settings` → 왼쪽 `Pages`
- `Branch`를 `main` / `/root` 로 선택하고 `Save`
- 1~2분 뒤 새로고침하면 `Your site is live at https://<아이디>.github.io/diet-tracker/` 주소가 뜸

**4. 폰에서 설치**
- 안드로이드 **크롬**으로 그 주소 열기
- 우측 상단 `⋮` 메뉴 → **`홈 화면에 추가`** (또는 `앱 설치`)
- 홈 화면에 초록 아이콘 생김. 탭하면 주소창 없는 전체화면 앱으로 실행됨

끝. 데이터(체크·걸음·체중)는 폰에 저장돼서 앱 닫아도 유지되고, 날짜 바뀌면 그날 기록으로 새로 시작함.

---

## 방법 B. 그냥 파일로 열기 (테스트용, 설치 안 됨)

`index.html`을 폰에 저장하고 크롬으로 열면 화면은 보임. 단 이 방식은 PWA 설치(홈 아이콘·오프라인)가 안 되고, 매니페스트/서비스워커도 작동 안 함. 진짜 앱처럼 쓰려면 방법 A로 웹에 올려야 함.

---

## 참고

- 이건 **PWA(설치형 웹앱)**라 안드로이드 홈 화면에서 아이콘으로 실행되지만, 런처에 얹는 "네이티브 위젯"(홈 화면 바탕에 상시 표시)은 아님. 그건 앱 개발(Kotlin)이 필요해서 완전히 다른 작업이야.
- 내용(끼니·매크로·목표) 바꾸고 싶으면 `index.html` 상단의 `MEALS`, `TARGETS` 부분만 고치면 됨. 수정해줄 수 있으니 말해.
- iOS도 동일하게 사파리에서 `공유 → 홈 화면에 추가`로 설치돼.
